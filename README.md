Program Latihan4.java bertujuan menghitung volume tabung menggunakan input jari-jari dan tinggi dari pengguna.
Perhitungannya dilakukan dengan rumus π × r² × t dan hasil akhirnya ditampilkan di layar.
Program menggunakan fungsi tambahan (method) agar kode lebih rapi dan modular.
